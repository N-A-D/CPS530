    Django installation instructions


    • Check if pip is installed
      On your local machine make sure python and pip is installed.
      If pip is not installed, follow these instructions
      Installing Django
      Open a shell prompt and enter: $ pip install django
      Create a project folder
      Create a folder for your project: $ mkdir projectName
      Create the Django Project
      Open a command prompt/terminal in the project folder and start the Django project.
      $ ctrl t
      $ django-admin startproject [projectName]
      Change directory
      Change directory into the project directory that was created previously.$ cd projectName
      Start development server
      To start the development server enter.$ python manage.py runserver
      Access the server
      Open a browser and enter the address to which the development server is being run. You can find the address in the output of running the previous step.

    • Foundation installation instructions


    • Installing with common package managers:
      You can install foundation using package managers it is available on. (npm, Bower, Meteor and Composer).
      npm- $ npm install foundation-sites
      pnpm -$ pnpm install foundation-sites
      yarn -$ yarn add foundation-sites
      bower -$ bower install foundation-sites
      gem -$ gem install foundation-rails
      meteor -$ meteor add zurb:foundation-sites
      php -$ php composer.phar require zurb/foundation
      NuGet -$ Install-Package foundation-sites

      What comes with the installation:
      |─ scss       Source Sass files. Use this folder as a load path in Sass.
      
├─ js         Source JavaScript files. If you're using a build system, make sure `foundation.core.js` is loaded first.
      
└─ dist       Compiled files:
      
├─ css        * Compiled CSS files. Includes minified and unminified files.
      
├─ js         * Concatenated JavaScript files. Includes minified and unminified files.
      
└─ plugins    * Standalone JavaScript plugins.
