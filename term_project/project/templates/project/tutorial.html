In this tutorial we will guide you through, how we created our webpage.
Prerequisite
  1. Ensure that Python 3 and Django for Python3 is installed.
  2. Create a project folder for your django project: django-admin startproject OurAnimeList
  3. Change directory into the project folder that was created from step (3).
    4. In Django, a website is composed of applications. The main site just references them.
  4. Type in the following to create an application with that project folder: python manage.py startapp animes
  5. In the same working directory, open OurAnimeList/settings.py with an editor of your choice.
  6. Find the INSTALLED_APPS list and add the following as an item in the list: 
    ‘animes’,
  7. In the same working directory as, open up OurAnimeList/urls.py
  8. At the top of the script file, import include from django.urls.
  9. Find the urlpatterns list and add the following as an item in the list: path(‘’, include(‘animes.urls’)),save and exit the file.
  10. Change directory to animes.
  11. Create a file called urls.py
  12. Open up urls.py and enter in the following code:
    from django.urls import path, include

urlpatters = [
    path('', views.index, name='index'),
]
    save and exit the file.
  13. Open up views.py with a text editor.
  14. In views.py enter the following code:
    from django.shortcuts import render

def index(request):
    return render(request, 'animes/showcase.html')
     save and exit the file.
