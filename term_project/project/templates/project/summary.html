Foundation 6 Framework

Foundation is an advance front-end framework created by web design company Zurb. It is a framework for designing beautiful responsive websites, apps and emails that works on all types of devices. Foundation provides HTML, CSS and JavaScript plugins for making website creation more attractive and fun. It is use by websites like Facebook, eBay, Adobe, EA, Amazon and Mozilla. Foundation began as an internal style guide in 2008 and then released as open-source in 2011 under the MIT license. Since then its popularity has been able to rival front-end frameworks like Bootstrap and Semantic-UI. Foundation provides layout templates for users, allowing them to create websites faster with less hassel. However, this framework is fairly complex and is more widely used by experience web designers for faster and creative designs. According to Sitepoint.com, Foundation is the 2nd most popularly used front-end framework of 2018, just losing out to Bootstrap. According to Google trends, of the five front-end frameworks Bootstrap, Semantic UI, Foundation, Material UI and Materialize, Foundation is the 2nd most popularly search framework since 2012.

Strenghts:
  • It has a very powerful grid system enabling the designers to push the limits of their creativity.
  • The styles are intentionally underdeveloped to allow for more creativity and flexibility by designers.
  • It uses Rem instead of pixels making it more flexible when dealing with different devices.
  • Services are offered for websites, apps and emails.
  • It comes with a large collection of styles that separates it from other front-end frameworks.
  • It offer layout templates for faster web development.
  • It offer four different types of installation to fit users development needs.
  • It offers a beautifully designed documentation with interactive tutorials.
Weaknesses:
  • Complexity makes it difficult for novice web designers.
  • Some installation file sizes are large.

Django Framework

Django is an open source and free high-level python framework that makes it easier for developers to write their apps. Originally developed by Adrian Holovaty and Simon Willison, Django is maintained by the Django Software Foundation.
Python is one of the most popular and top-notch programming languages as it is easy to learn, versatile, and fast to write. It is most popular in Cuba and China.
